These are some theme and style guidelines to keep the Flaggi documentation clear, consistent, and easy to read.

# Colors

- `#0B0D14` for dark backgrounds
- `#3F80F0` for the Flaggi blue
- `#E02F40` for the Flaggi red

# Fonts

- [Pixelify Sans](https://fonts.google.com/specimen/Pixelify+Sans): In game pixalated font
- [Inter](https://fonts.google.com/specimen/Inter): For documentation and UI text or any other web content (dev console, website, etc.)

:::note[Note]
There are older styles used in the game UI, banners and similar. These will get updated in the future to match the new branding (like the webstite or devconsole).
:::
